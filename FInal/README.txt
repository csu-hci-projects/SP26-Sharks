VR Basketball Game

Author:
David Samuels

Project Overview:
This project is a Virtual Reality basketball game created in Unity using VR interaction systems and tested using a Meta Quest 2 headset. The purpose of the project is to create an immersive basketball shooting experience where the player can interact with a virtual basketball court and shoot basketballs using VR controller input.

Features:

* VR basketball shooting mechanics
* Physics-based basketball movement
* Interactive VR environment
* VR headset and controller support
* Basketball court environment
* Real-time gameplay interaction

Software and Tools Used:

* Unity Game Engine
* OpenXR
* XR Interaction Toolkit
* Meta Quest Link
* Visual Studio Code

Hardware Requirements:

* Meta Quest 2
* VR-ready computer
* USB-C cable or Quest Link connection

How to Run the Project:

1. Install Unity Hub and Unity Editor.
2. Open the project folder in Unity.
3. Connect the Meta Quest 2 headset using Quest Link.
4. Open the basketball scene inside Unity.
5. Press the Play button in Unity.
6. Put on the headset and begin playing the VR basketball game.

Controls:

* Move headset to look around the environment.
* Use VR controllers to interact with objects.
* Press controller buttons to shoot or interact with the basketball.

Project Structure:

* Assets: Game assets, scripts, scenes, prefabs, materials, and audio.
* Packages: Unity package dependencies.
* ProjectSettings: Unity configuration settings.

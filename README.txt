VR Basketball Game

Author:
David Samuels and Tigist Kelefew

Project Overview:
This project is a Virtual Reality basketball game developed in Unity and tested using a Meta Quest 2 headset. The goal of the project is to create an immersive VR basketball experience where players can interact with a virtual basketball court and shoot basketballs using VR controller input and motion-based interaction.

Features:

* VR basketball shooting mechanics
* Physics-based basketball movement
* Interactive VR environment
* Basketball court and hoop system
* Real-time score tracking
* VR headset and controller support
* Unity XR interaction support

Software and Tools Used:

* Unity Game Engine
* OpenXR
* XR Interaction Toolkit
* Visual Studio Code
* GitHub
* Meta Quest Link

Hardware Requirements:

* Meta Quest 2 headset
* VR-ready computer
* USB-C cable or Quest Link connection

How to Run the Project:

1. Install Unity Hub and Unity Editor.
2. Download or clone the project repository.
3. Open the project folder in Unity.
4. Connect the Meta Quest 2 headset using Quest Link.
5. Open the main basketball scene inside Unity.
6. Press the Play button in Unity.
7. Put on the headset and interact with the VR basketball environment.

Controls:

* Move headset to look around the environment.
* Use VR controllers to interact with objects.
* Use controller input to shoot or interact with the basketball.

Project Structure:

* Assets: Contains scenes, scripts, models, materials, audio, and prefabs.
* Packages: Unity package dependencies and XR packages.
* ProjectSettings: Unity project configuration settings.

Project Scripts:

* BallLauncher.cs: Handles basketball launching and shooting mechanics.
* ScoreKeeper.cs: Tracks and updates the player's score.
* DetectScoring.cs: Detects successful basketball shots.
* LevelManager.cs: Controls scene transitions and game flow.
* ScoreDisplay.cs: Updates score UI text during gameplay.
* TimeRemainingDisplay.cs: Displays remaining game time.
